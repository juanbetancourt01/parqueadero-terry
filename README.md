## Proyectos

### Parqueadero
Sistema de gestión de parqueadero en C++ con mapa visual 16x16,
animaciones de movimiento, dos niveles, y sistema de cobro por tiempo.

*Innovaciones implementadas:*
- Matriz de vías independiente para circulación organizada
- Dos niveles con reglas distintas por tipo de vehículo
- Animaciones de movimiento celda por celda
- Diferenciación visual con emojis y colores por tipo
- Historial de pagos por sesión
